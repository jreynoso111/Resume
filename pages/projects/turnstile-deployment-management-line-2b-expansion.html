Turnstile Deployment Management for Metro Santo Domingo Line 2B Expansion

Operational rollout of new station access barriers across Metro Santo Domingo Line 2B, coordinating civil, electrical, software, and station operations teams under a compressed deployment window.

Project Overview

I coordinated deployment operations among Metro Santo Domingo operations leadership, civil works crews, electrification teams, the turnstile supplier/installer, and the software integration partner. The project required synchronized night windows, strict station handover timing, and cross-team dependency control so each site could reopen for morning service without interruption.

Main Features

  • Multi-entity deployment planning with unified station-by-station execution boards.
  • Replacement strategy for mixed barrier configurations: start, middle, and end turnstile types.
  • Overnight replacement cycles to avoid daytime passenger-service impact.
  • Integrated testing flow between hardware installation, electrification, and software validation.
  • Rapid issue escalation protocol to keep night shifts on schedule and reopen stations on time.

Key Components

Station Deployment Sequencing

Defined nightly cutover plans per station, including pre-work staging, access windows, equipment swaps, and morning readiness checkpoints.

Mixed Turnstile Type Management

Coordinated installation logic for start, middle, and end barrier positions, aligning physical layout, cabling, and control logic for each configuration.

Hardware + Software Go-Live

Managed handoffs between installer and software integrator so every newly deployed barrier passed acceptance checks before station handback.

Operational Impact

Turnstiles were replaced overnight at each station, minimizing service downtime and preserving daytime operations. In just one and a half weeks, deployment teams updated barriers across 10 stations on Line 2B while maintaining consistent reopening readiness each morning.
